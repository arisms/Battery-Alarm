Battery Alarm

View project on GitHub

Battery Alarm allows users to create their custom notifications for their device's battery status. Users can create, edit and delete alarms by choosing the battery level(%) and charging status.

They can also select the way they want to be notified each time (notification sound, volume and vibration). Each alarm can be activated/deactivated.

Whenever the battery meets the condition (level and charging status) of an active alarm, the user will receive a notification.

The application utilizes the following SDK tools:

  • Service for creating the notifications in the background when needed
  • AlarmManager to trigger the service in regular intervals
  • RecyclerView with a custom Adapter to display the created alarms
  • AppCompatActivity to allow use of Material Design for older versions of Android
  • GSON/JSON to store the list of alarms in the SharedPreferences file
